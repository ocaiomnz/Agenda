#Projeto base de agendamento ---
==========================================================
Funções principais:
1- Alocar horário no bloco
2- Verificar se está disponível
3- Confirmar a alocação
4- Possibilidade de cancelamento
==========================================================
#Definição das características
==========================================================
-Blocos de 30 min
-Considerar 24h de abrangência
-Não podem existir mais de um agendamento no mesmo horário
-Precisa do Nome completo para agendar
-Após ser cancelado, o horário fica disponível novamente
==========================================================
#Qual a estratégia?
==========================================================
1 - Coletar dados do usuário
2 - Escolher a operação desejada
3 - Criar Banco de marcação
4 - Ao ser selecionado o bloco sai do Banco e fica associado ao usuário
5 - Em caso de consulta é mostrada a lista com os horários (vagos)
6 - Em caso de cancelamento, o bloco volta a lista e usa-se o Sort para reorganizar em ordem
